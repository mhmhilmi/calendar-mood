# Blocalm Flutter App

Flutter app scaffold for Blocalm.

## Setup

If this folder was not generated by `flutter create`, run the Flutter platform generator from inside `app/` before the first full build:

```bash
flutter create --platforms=android,ios,windows,macos,linux .
```

Then fetch dependencies:

```bash
flutter pub get
```

Run with a local backend:

```bash
flutter run --dart-define=API_BASE_URL=http://localhost:8000/api/v1
```

On a physical phone, replace `localhost` with the computer's LAN IP address.
